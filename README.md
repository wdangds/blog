# Wan Dang Portfolio

This repository contains my Quarto-based portfolio site.

## Local Development

Render the site:

```bash
quarto render
```

Preview the site locally:

```bash
quarto preview
```

## Deployment

Pushing to the `v4` branch triggers the GitHub Actions workflow in `.github/workflows/publish.yml`, which publishes the site to GitHub Pages.

## Archive

The previous content source is preserved in `archive/quartz-content/`.
